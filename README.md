# A4AI.ca - Static Website

A professional static website for A4AI.ca (AI for All), showcasing custom AI solutions for small enterprises.

## Overview

A4AI.ca specializes in providing custom AI solutions tailored for small businesses, including chatbots, predictive analytics, process automation, and machine learning models.

## Website Structure

### Pages
- **index.html** - Homepage featuring services and company overview
- **support.html** - Support center with contact forms and FAQ
- **privacy.html** - Comprehensive privacy policy

### Features
- Modern, responsive design with gradient purple theme
- Mobile-friendly navigation
- Professional styling with CSS3 animations
- Contact forms and support options
- Comprehensive FAQ section
- Clean URL routing via Vercel

## Deployment

This site is deployed on Vercel with clean URLs enabled.

### Routes
- `/` - Homepage
- `/support` - Support page
- `/privacy` - Privacy policy

### Local Development

Simply open `index.html` in a web browser, or serve with any static file server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

## Configuration Files

- **vercel.json** - Vercel deployment configuration with clean URLs
- **.gitignore** - Git ignore patterns for IDE, OS, and build files
